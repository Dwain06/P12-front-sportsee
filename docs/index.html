2. How to run this project

2.1 Back end

Clone this repo https://github.com/OpenClassrooms-Student-Center/P9-front-end-dashboard

Follow instructions on readme file to run this project. Choose to run localhost on port 3000.

2.2 Front end

Prerequisites

  • NodeJS (version 12.18)
  • NPM

Launching the project

  • Fork the repository
  • Clone it on your computer.
  • npm install command will allow you to install the dependencies.
  • npm start command will allow you to run in your brother. Choose localhost on port 3001 to run both front and back.
